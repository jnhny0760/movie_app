AJAX(Aysnchoronous Javascript and XML)은 웹페이지의 이동 없이 현재 페이지에서 필요한 테이터만 
로드 또는 전송하는 기술을 말한다.
보통의 경우 데이터 처리는 요청 받는 순서대로 진행하지만 AJAX는 순차적으로 진행하지 않는다. 
이러한 방식을 '비동기식 방식'이라고 부른다.
JSON은 자바그크립트 객에 리터럴(키-값) 형식의 구문으로 구성된 데이터이다.
JSON은 외부로 전송하여 데이터를 처리 할 수 도 있고 내부로 로드하여 처리할 수도 있다.


$.ajax()메소드를 이용하여 외부에 있는 JSON파일을 로드하여 처리하는 방법 실습.

json,xml,hrml
xhtml=xml+html


xml(eXtensible Markup Language)태그의 목적은 해당 태그가 가지고 있는 데이터의 종류를 설명한다. 
데이터 형식을 기술하기 위한 마트업 언어이다. 

태그 작성법
대소문자 구분
공백x
첫글자는 한글, 영어, _, 사용가능
-,_는 태그 중간에 올 수 있음
